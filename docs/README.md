# Documentation

This directory contains the core reference documentation for the replication project.

The file [nsdi21-ghigoff.pdf](file:///home/olly/UNI/NetCmp/BMC_Check/docs/nsdi21-ghigoff.pdf) represents the original NSDI '21 paper titled "BMC: Accelerating Memcached using Safe In-Kernel Caching" authored by Yoann Ghigoff, Nicolas Jégou, and Gilles Muller.

The file [REPORT_v3.1.pdf](file:///home/olly/UNI/NetCmp/BMC_Check/docs/REPORT_v3.1.pdf) represents our comprehensive final project report, detailing the environment setup on CloudLab, the quantitative replication of the paper's main targets, our seven original exploratory stress testing campaigns, the technical compilation and runtime hurdles we resolved, and our final engineering conclusions.
