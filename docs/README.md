# Documentation

This directory contains the core reference documentation we used and produced during the replication project.

The file [nsdi21-ghigoff.pdf](file:///home/olly/UNI/NetCmp/BMC_Check/docs/nsdi21-ghigoff.pdf) represents the original NSDI '21 paper titled "BMC: Accelerating Memcached using Safe In-Kernel Caching" authored by Yoann Ghigoff, Nicolas Jégou, and Gilles Muller.

The file [REPORT_V_3_2_ON.pdf](file:///home/olly/UNI/NetCmp/BMC_Check/docs/REPORT_V_3_2_ON.pdf) represents the final compiled report we wrote, detailing the environment setup we configured on CloudLab, the quantitative replication we achieved for the paper's main targets, the seven exploratory stress testing campaigns we designed, the technical hurdles we resolved, and our engineering conclusions.

The files [configuration_sut.md](file:///home/olly/UNI/NetCmp/BMC_Check/docs/configuration_sut.md) and [configuration_client.md](file:///home/olly/UNI/NetCmp/BMC_Check/docs/configuration_client.md) contain the detailed step-by-step setup, dependencies installation, compilation instructions, and networking commands we executed on the SUT and client nodes to prepare the bare-metal CloudLab environments.
