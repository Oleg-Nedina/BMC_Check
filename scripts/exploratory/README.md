# Exploratory Stress Testing Scripts

This directory houses the custom stress-testing scripts we designed to isolate the performance envelopes, verification failures, and physical limits of our BMC implementation.

Under closed-loop workloads, we use the [stress_cache_warmup.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_cache_warmup.sh) script to profile our initial hit rate trajectory immediately following a cold-start reset. We run the [stress_tail_latency.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_tail_latency.sh) script to evaluate memaslap response time percentiles up to the P999 threshold under varying skew parameterizations. We execute the [stress_multiget_limit.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_multiget_limit.sh) script to sweep key numbers in multi-GET packets and detect driver-level descriptor drops. We run the [stress_hot_invalidation.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_hot_invalidation.sh) script to sweep write command ratios to measure cache invalidation overheads. We also run the [stress_working_set.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_working_set.sh) script to sweep database capacity limits to validate our memory partition assumptions.

Under open-loop workloads, we run the [stress_spinlock_scaling.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_spinlock_scaling.sh) script to force SUT core concurrency under a saturating hot-key flood to profile CPU serialization on BPF spinlocks. We execute the [stress_noise_flood.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_noise_flood.sh) script to flood SUT non-target ports with background UDP traffic to quantify the pre-stack packet inspection tax. We also run the [stress_payload_boundary.sh](file:///home/olly/UNI/NetCmp/BMC_Check/scripts/exploratory/stress_payload_boundary.sh) script to sweep payload thresholds under multiple core counts to profile our worst-case parser overhead.
